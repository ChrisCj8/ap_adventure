<p>The Map Tab is where you can set settings specific to the current map.</p>
<p>Many of these settings can also be set in the group tab, but everything set in here will overrule whatever has been set in the group settings.</p>
<p>All settings with light blue text are currently being inherited from the group settings.</p>
<p>Some settings have help icons next to them that you can click for more infomation.</p>
